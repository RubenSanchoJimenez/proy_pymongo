Estos son los FrameWorks usados

Angular + Django + NodeJS

Angular

Con esto framework hemos controlado el front o la experiencia de usuario, creando un entorno visual y atractivo para poder operar de forma visual. En este código se usa HTML, CSS (Bootstrap) y TypeScript para poder ejecutarse.

Django

En Django se ha realizado una arquitectura por end-points en las que se reciben llamadas del front las procesa en base a lo que solicita y se llama al servidor que en este caso es MongoDB y devuelve una respuesta. En este framework se usa Python para la lógica.

NodeJS

En nuestro proyecto, Node.js se usa como intermediario para gestionar solicitudes asíncronas entre Angular y Django, optimizando el flujo de datos entre el cliente y el servidor. Además, Node.js permite realizar tareas de procesamiento en segundo plano y gestionar conexiones en tiempo real si quieres añadir interactividad dinámica o notificaciones en tiempo real.
